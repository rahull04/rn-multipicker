# rn-multipicker

This package will provide features to select and search multiple dropdown items gracefully

## Installation

```sh
npm install rn-multipicker
```

## Usage

```js
import { multiply } from 'rn-multipicker';

// ...

const result = await multiply(3, 7);
```
